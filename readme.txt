Extends WordPress and gives WordPress the ability to device sessions aloowing for mobile application 
login and registration on a mobile application with the need for HTTP for webviews.

== Description ==

WordPress Mobile Oauth gives your WordPress site the ability store mobile user sessions allowing your 
mobile application users to register and log into your mobile application.
